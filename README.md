# Excel Sales Dashboard with VBA Automation

## Overview

This project is an interactive Sales Dashboard developed in Microsoft Excel using Pivot Tables, Pivot Charts, Excel formulas, and VBA Automation.

The dashboard allows users to analyze sales performance across multiple cities through automated navigation and dynamic reports.

## Features

- Interactive Sales Dashboard
- City-wise Sales Analysis
- Pivot Tables & Pivot Charts
- Dashboard Navigation using VBA
- Macro Recording + Custom VBA Code
- Automated Dashboard Linking & Unlinking
- Dynamic Sales Reports

## Tools Used

- Microsoft Excel
- VBA
- Pivot Tables
- Pivot Charts

## Skills

- Excel Automation
- Data Analysis
- Dashboard Design
- VBA Programming
- Reporting
