# 📊 HR Analytics Dashboard (Microsoft Excel)

## 📌 Overview

This project is an interactive **HR Analytics Dashboard** developed in **Microsoft Excel** to analyze workforce metrics, employee attrition, headcount trends, diversity, and reporting structures.

The dashboard enables HR teams and managers to monitor key performance indicators (KPIs) through dynamic visualizations, Pivot Tables, Pivot Charts, Slicers, and Excel automation.

## 📊 Dashboard Components

### 📌 Head Count Growth
Tracks yearly employee onboarding, attrition, transfers, and overall workforce growth.

### 📌 Population Diversity
Displays gender distribution across the organization.

### 📌 Employee Levels
Visualizes employees across Management, Professional, Trainee, and Contract roles.

### 📌 Department-wise Count
Shows employee distribution across different departments.

### 📌 Reporting Manager Analysis
Displays headcount managed by each reporting manager.

### 📌 Employee Status
Compares Permanent and Contract employees.

### 📌 Employee Stability
Analyzes employee tenure across different experience ranges.
